# Technique taxonomy

Full name index by category. Consult only when the task needs a named technique
outside SKILL.md's Priority/Conditional/Experimental tiers — most real usage is
covered there and never reaches this file.

**Basics**: Zero-shot, Few-shot, One-shot, Role/Persona, Instruction, Context, Delimiters (XML/MD/tags), Output Formatting, Negative Prompting, Templates.
**Reasoning**: Chain-of-Thought (CoT), Zero/Few-shot CoT, Self-Consistency, Least-to-Most, Tree of Thoughts, Graph of Thoughts, Skeleton-of-Thought, Plan-and-Solve, Analogical, Step-back, Thread-of-Thought.
**Decomposition**: Prompt Chaining, Task Decomposition, Sequential/Recursive Prompting, Progressive Refinement.
**Self-correction**: Self-Refine, Reflexion, Self-Critique, Chain-of-Verification (CoVe), ReAct, PAL.
**Multiple answers**: Best-of-N, Majority Voting, Ensemble, Debate, Multi-Agent.
**Tools**: Function Calling, Tool Use, RAG, Context Injection, Grounding, External Memory.
**Optimization**: APE, Prompt Optimization/Compression, Meta Prompting, Prompt Generation/Mutation, Prefix/P-Tuning.
**Code**: Program-of-Thought, Code Prompting, Execution-based Prompting.
**Agents**: Agent Prompting, Planning Agents, Tool-Augmented Agents, Multi-Agent Systems.
**Safety**: Defensive Prompting, Jailbreak Resistance, Prompt Injection Defense, Input Sanitization, Guardrails.
**Evaluation**: Prompt/Benchmark/Automatic/Human Evaluation, A/B Testing.
**Multimodal**: Image/Vision/Multimodal Prompting, Image+Text Reasoning.
**Experimental**: Active Prompt, DSP, Generated Knowledge, Faithful CoT, Tree Search, Selection-Inference, Emotion Prompting.

Base: The Prompt Report (arXiv:2406.06608) + official vendor docs — not own evaluation.
